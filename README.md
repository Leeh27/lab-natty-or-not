⚡ Projeto: Transformação em Cena de Harry Potter com IA

📝 Descrição

Este projeto consiste na transformação de uma foto pessoal em uma cena inspirada no universo de Harry Potter, utilizando inteligência artificial para criar um visual cinematográfico e imersivo.

🚀 Tecnologias Utilizadas

-  IA Generativa **[ChatGPT](https://chat.openai.com)** (para criação de prompts)
-  GEMINI **[ChatGPT](https://gemini.google.com))** para geração/edição de imagem

🎨 Processo de Criação

Utilizei uma foto original minha e apliquei IA para transformá-la em uma cena temática de Harry Potter.
Foram feitos ajustes como:

- Mudança de ambiente
- Estilo cinematográfico
- Iluminação dramática
- Elementos visuais inspirados no universo mágico
